# Coursework Submission Tracker

Coursework Submission Tracker is a TypeScript project for managing courses and GitHub-based coursework submissions across student, instructor, and admin roles, with typed entities for users, courses, submissions, status tracking, and API responses.

## Core Entities

- `User`
- `Course`
- `Submission`
- `SubmissionStatus`
- `ApiResponse<T>`

## Advanced TypeScript Features Used

- String enums
- Generic interfaces
- Generic functions
- Utility types: `Partial`, `Pick`, `Omit`, and `Record`
- Indexed access types
- Union types
- Intersection types
- Type-only imports

## Installation and Execution

```bash
npm install
npm run start
```

## Type Check

```bash
npm run typecheck
```
